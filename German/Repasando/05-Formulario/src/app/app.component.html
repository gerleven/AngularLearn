2- Acceso a los controles desde el código mediante su "Template reference variable" para leer y modificar su estado
Un elemento del template puede asociarse a una variable
Podemos usar esa variable en el código del template para manejar ese elemento

podemos usar el #ref del input dentro del codigo html para usar los metodos del API DOM de ese elemento html
Por ejemplo dentro del (click) del button para pasar el valor de inputRef.value a la funcion addItem() haciendo addItem(inputRef.value)

Si queremos renderizar el valor de inputRef.value asi { { inputRef.value }} vamos a ver que no se actualiza en tiempo real como el [(NgModel)] justamente porque no es un binding bidireccional : "{{inputRef.value}}"

Styles

Hay 3 maneras equivlentes de escribir una misma propiedad de estilo

por ejemplo para 'backgraound color' podemos escribir: backgraound-color = backgraound-Color = backgraoundColor

Ejemplos:

span{border-style: solid;}

style="border-color: red;" style="border-Color: aqua;" style="borderColor: yellow;" [style.border-color]= "'green'" [style.border-Color]= "'orange'" [style.borderColor]= "'pink'"


{{"[style.property]='styleValue':"}}

{{"<p [style.color] = variableConStyle> Text </p>"}}

{{"<p [style.color] = “ variableConStyle ” > Text </p>"}}

{{"<p [style.color] = “ 'string' ”> Text </p>"}}

{{"<p [style.color] = “funcion()”> Text </p>"}}

{{"<p [style.color] = funcion()> Text </p>"}}


{{"[ngStyle]='styleObject':"}}

[ngStyle] = "myObjectMapStyle" = {"color": "red", "font-size": "smaller" }

[ngStyle] = "getMyObjectMapStyle()" = {"color": "white", "font-size": "smaller", "backgraound-Color": variableConStyle ? blue : red }


cuando es una property con unidades:

{{"<p [style.fontSize.px] = “ 25 ”> Text </p>"}}

{{"<p [style.fontSize.%] = “ 95 ”> Text </p>"}}
